Pretext-powered portfolio ideas

3 approaches for making your single-page bio more interactive

A

Expandable Bio Cards

Each line item (Kalshi, UWaterloo, Traverse, etc.) becomes clickable. Clicking smoothly expands to reveal more detail — what you built, what you learned, key metrics. Pretext pre-calculates the expanded text height so the animation is buttery smooth with zero layout jank.

Think: your resume becomes an interactive story. Visitors discover depth on demand.

B

Animated Page Load Cascade

Instead of all text appearing at once, sections smoothly cascade in from zero height to their full measured height. Pretext calculates each section's final height upfront, so the staggered reveal animation is perfectly choreographed — no content jumping.

Think: each section unfolds like a letter being opened, timed to the WebGL cursor effect.

Load sequence
1. "Lance Yan" + icons fade in (0ms)
2. "Currently" section grows from 0 → full height (200ms)
3. "Previously" section grows from 0 → full height (400ms)
4. "Building" section grows from 0 → full height (600ms)
Each section's target height is pre-calculated by pretext — no DOM measurement needed, so animations start instantly without a layout pass.
C

Both A + B Combined

Page loads with the cascade animation, then each item is expandable for detail. The site feels alive — it breathes on load, then responds to curiosity.
